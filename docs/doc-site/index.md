# TouchFish-Docs 文档站点

**TouchFish-Docs** 文档站点是由 [@ILoveScratch2](https://github.com/ILoveScratch2) 和 TouchFish 社区 为 [TouchFish](https://github.com/2044-space-elevator/TouchFish) 聊天工具打造的官方文档网站。

基于 [VitePress](https://vitepress.dev/) 构建，提供使用指南、安装教程和常见问题解答，帮助更好地了解和使用 TouchFish。

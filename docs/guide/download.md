# 下载 TouchFish

欢迎下载 TouchFish！根据您的需求和平台，可以选择不同的发行版本。

## 版本选择指南

> [!WARNING]
> 大部分发行版**不支持 TouchFish V4 协议**，仅支持 TouchFish V3 及更早版本。

TouchFish 拥有多个发行版本，各有特色：

- **LTS** - 官方根版本，功能完善，稳定可靠
- **Astra** - 全平台支持，UI 精美，支持 Markdown 和 LaTeX
- **UI Remake (TFUR) (不支持TouchFish V4)** - 现代化 UI，支持 Markdown、代码高亮和部分 LaTeX
- **Pro(不支持TouchFish V4)** - 支持 Markdown 和 LaTeX，有用户高亮功能
- **More (Lite)(不支持TouchFish V4)** - 更好的性能，更快的下载速度
- **Plus(不支持TouchFish V4)** - 有更多功能（不兼容）
- **For Android(不支持TouchFish V4)** - 安卓移动端（已不推荐）
- **TouchMouse(不支持TouchFish V4)** - 基于 TouchFish 的聊天项目，进行优化和改进（不兼容）

### 其他

- **Cloud Studio Chat (CSC)** - 高性能聊天软件，使用 C++ 编写，兼容性强（不兼容）

## 平台特殊说明

> 此部分针对LTS版本

### Windows 用户

::: warning Windows 7 用户注意
如果您使用 Windows 7 或更低版本系统，可能需要额外安装 DLL 文件才能正常运行。
:::

大部分版本都提供了预编译的 `.exe` 文件，下载后可直接运行。

### MacOS 用户

::: warning 安全设置
该软件需要打开任意来源以正常运行：

1. 打开 **系统设置**
2. 进入 **安全性与隐私** → **安全性**
3. 在 **允许以下来源的应用程序** 中点击 `App Store 与已知开发者`
4. 选择 **任何来源**
:::

部分版本需要自行编译，请参考各版本的 GitHub 页面说明。

### Linux 用户

::: warning UI 要求
大部分版本仅支持在具有 UI 界面的 Linux 发行版中运行。

如需在无 UI 界面的 Linux 发行版中运行，请使用 **CLI 版本**。
:::

大部分版本需要自行编译或使用 Python 源码运行。

## 源码运行

如果您的平台没有预编译包，可以通过 Python 运行源码：

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行
```bash
python LTS.py
```

## 版本兼容性说明

**兼容版本（V3）：** LTS, CLI, UI Remake, Pro, More, For Android, Astra

**兼容版本（V4）：** LTS, Astra

这些版本之间可以互相连接，服务器端和客户端可以使用不同的兼容版本。

**不兼容版本（V3）：** Plus, TouchMouse

Plus 版本不与其他版本兼容，且不同 Plus 版本之间也不兼容。
TouchMouse 版本也不与LTS兼容。

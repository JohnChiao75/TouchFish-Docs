# 关于 TouchFish

**TouchFish** 是一款专为局域网环境打造的轻量级聊天软件

### 诞生背景

在学校机房等断网环境下，传统的 QQ、微信等聊天工具无法使用。TouchFish 应运而生，它只需要局域网就能工作，也可在公网使用。

### 设计理念

- **简单** - 功能精简，专注聊天本身
- **轻量** - 体积小，占用资源少
- **易用** - 三步搞定，小白也能用
- **开源** - 代码开放，欢迎改进


主仓库：[GitHub 2044-space-elevator/TouchFish](https://github.com/2044-space-elevator/TouchFish)

## 重要提示

::: warning Windows 7 及以下系统
使用 Windows 7（及更早版本）的用户可能需要额外安装 DLL 文件才能正常运行。
:::

::: warning macOS 用户注意
需要开启"允许任何来源"：

系统偏好设置 → 安全性与隐私 → 通用 → 允许从以下位置下载的应用 → 任何来源
:::

::: warning Linux 用户注意
本软件需要 GUI 界面才能运行。

如果你使用的是无 GUI 的服务器，请使用 [CLI 命令行版本](./guide/download#cli)。
:::

感谢你对 TouchFish 的关注！如果你想提交 PR，**请务必先阅读** [贡献者须知](https://github.com/2044-space-elevator/TouchFish/blob/main/CONTRIBUTING.md)，避免浪费时间精力。

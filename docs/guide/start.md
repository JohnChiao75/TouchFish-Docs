# 快速开始

## 作为服务端（创建聊天室）

### 获取内网 IP 地址

Windows 用户：打开命令提示符，执行 `ipconfig`，查找 `IPv4 地址`

Linux/macOS 用户：打开终端，执行 `ip a` 或 `ifconfig`

### 检查端口可用性

Windows：`netstat -an | findstr <端口号>`（无输出表示端口空闲）

Linux/macOS：`netstat -an | grep <端口号> 或 lsof -i :<端口号>`

### 启动服务器

1. 运行 TouchFish 程序
2. 按下 <Enter> 键开始配置
3. 选择启动方式为 `Server`
4. 输入您的内网 IP 地址
5. 指定一个可用的端口号
6. 设置服务端昵称（将在聊天室中显示）
7. 设置最大用户数（建议不超过 128）
8. 将 IP 地址和端口信息分享给其他用户

### 后续启动
运行程序后，根据提示按下 <Ctrl + C> (Windows/Linux) 或 <Ctrl + D> (macOS)

程序将自动使用上次的配置启动服务器

## 作为客户端（加入聊天室）

### 首次连接

1. 运行 TouchFish 程序
2. 按下 Enter 键开始配置
3. 选择启动方式为 Client
4. 输入服务器提供的 IP 地址
5. 输入服务器提供的端口号
6. 设置您的个人昵称

### 后续连接

运行程序后，根据提示按下 <Ctrl + C> (Windows/Linux) 或 <Ctrl + D> (macOS)

程序将自动使用上次的配置连接服务器

## 系统要求

### Windows

Windows 10 或更新版本：支持

Windows 8.1 及更早版本：可能需要安装额外的 DLL 文件

权限要求：如果提示文件写入失败，请以管理员身份重新运行

### macOS

支持版本：macOS 10.8 Mountain Lion 及以上

需要启用"任何来源"应用运行权限：

打开"系统设置"

进入"安全性与隐私"

在"安全性"选项卡中

在"允许以下来源的应用程序"部分选择"任何来源"

### Linux

无特殊限制

服务端部署建议使用源代码版本而非编译后的二进制文件
